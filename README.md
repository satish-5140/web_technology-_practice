# Web Technology Practice

A simple HTML practice repository for learning and experimenting with core web technology concepts.

## Project overview

This repository contains example HTML files created while practicing basic HTML tags, page structure, and multimedia embedding.

## Files included

- `colrs.html` — practice with color styling and text formatting
- `heading_tags_practice.html` — practice using heading tags (`<h1>` through `<h6>`)
- `multimedia tags-practice.html` — practice embedding audio, video, and image media
- `tags_practice.html` — general HTML tag practice and layout examples

## How to use

1. Open any of the `.html` files in a web browser.
2. Review the markup to learn how different HTML tags are used.
3. Edit the files and refresh the browser to see changes.
